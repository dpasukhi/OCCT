# ODE Cap'n Proto Schemas

This directory contains Cap'n Proto schema files (`.capnp`) for the ODE (OpenCascade Data Exchange) format.

## Schema Files

The following schema files define the ODE format structure:

1. **ode.capnp** - Base types (Vec3, Vec2, Quaternion, Transform, ObjectRef)
2. **surfaces.capnp** - All 12 surface types (Plane, Cylinder, BSpline, etc.)
3. **curves3d.capnp** - All 11 3D curve types (Line, Circle, BSpline, etc.)
4. **curves2d.capnp** - All 11 2D curve types (Line2d, Circle2d, BSpline2d, etc.)
5. **topology.capnp** - TopoDS topology structures (Shape, Vertex, Edge, Face, etc.)
6. **triangulation.capnp** - Poly_Triangulation tessellation data
7. **polygons.capnp** - Poly_Polygon3D and Poly_Polygon2D data

## Schema Compilation

These schemas are automatically compiled to C++ code during the build process:
- Input: `resources/ODE/*.capnp`
- Output: `build/src/DataExchange/TKDE/ODE/schemas/*.capnp.h` and `*.capnp.c++`

The compilation is configured in `src/DataExchange/TKDE/ODE/CMakeLists.txt`.

## Schema IDs

Each schema file must have a unique ID generated by:
```bash
capnp id
```
